# README

# リポジトリ名
anime-manga-ranking

## アプリ名
【投票制】アニメ・マンガランキング

## 使い方

1. ランキング一覧では投稿された作品をランキング形式で閲覧できます。
2. ユーザーの新規登録＆ログインを行うことで、作品の追加・投票・コメント機能を利用することができます。
3. 投票後は画面右下の「投票反映」を押下することで、投票結果がランキングに反映されます。
